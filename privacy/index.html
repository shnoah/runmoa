RunMoa 개인정보처리방침

최종 업데이트: 2026년 6월 18일

RunMoa는 Apple 건강에 저장된 러닝 기록을 iPhone에서 러너 중심으로 정리해 보여주는 앱입니다.

개요

RunMoa는 계정 생성을 요구하지 않으며 광고 SDK나 소셜 추적 기능을 포함하지 않습니다.

Apple 건강 러닝 기록, 경로 좌표, 심박 샘플, 러닝 메모, 신발 데이터는 개발자 서버로 업로드하지 않습니다.

앱 안정성과 사용 흐름을 이해하기 위해 Firebase Analytics와 Firebase Crashlytics를 사용합니다.

읽는 건강 데이터

사용자가 Apple 건강 접근을 허용하면 RunMoa는 다음 항목을 읽을 수 있습니다.

  • 러닝 워크아웃
  • 러닝 경로
  • 심박수와 안정시 심박수
  • VO2 Max
  • 걷기/달리기 거리와 걸음 수
  • 지원되는 iOS/watchOS의 운동 노력 데이터

이 데이터는 iPhone 안에서 러닝 요약, 러닝 컨디션, 심박 존, 상세 차트, 개인 기록, 신발 연결, 공유 이미지 표시를 만드는 데 사용됩니다.

기기 안에 저장되는 데이터

RunMoa는 앱 설정, 심박 존 설정, 신발 정보와 표시 순서, 러닝-신발 연결, 사용자가 작성한 러닝 메모, 개인 기록 검토 결과, 평균 심박/케이던스/상승 고도/상세 차트 같은 파생 요약과 캐시를 iPhone 앱 컨테이너에 저장합니다.

이 로컬 보조 파일에는 iOS 파일 보호를 적용하고 자동 iCloud/Finder 백업 대상에서 제외되도록 저장합니다.

Firebase 통계와 크래시 진단

RunMoa는 TestFlight와 App Store 배포 빌드에서 Firebase Analytics와 Firebase Crashlytics를 사용합니다. 이 서비스는 앱 사용 통계, 크래시 보고서, 스택 트레이스, 기기 및 운영체제 정보, 앱 버전, 진단 메타데이터를 수집할 수 있습니다.

RunMoa는 Apple 건강 운동 세부 정보, 경로 좌표, 심박 샘플, 러닝 메모, 신발 데이터, 사용자 ID를 Firebase 보고서에 직접 첨부하지 않습니다.

사용자가 직접 내보내는 데이터

RunMoa는 사용자가 명시적으로 선택한 경우에만 데이터를 외부 앱으로 전달합니다.

  • 생성한 PNG 공유 이미지를 클립보드에 복사
  • 생성한 PNG 공유 이미지를 파일로 저장
  • 신발 정보, 표시 순서, 러닝-신발 연결, 메모, 상세 첫 카드 지표 설정만 담은 개인 데이터 백업 JSON 공유

사용자 제어

  • Apple 건강 접근 권한은 설정 앱 > 건강 > 데이터 접근 및 기기 > RunMoa에서 변경할 수 있습니다.
  • 앱 설정에서 신발 데이터 삭제, 개인 데이터 백업 내보내기/가져오기, 분석 캐시 삭제를 할 수 있습니다.
  • 앱을 삭제하면 앱 컨테이너에 저장된 로컬 보조 데이터도 제거됩니다.
